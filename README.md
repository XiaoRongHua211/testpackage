# My Unity Package

A sample Unity package for demonstration purposes.

## Installation via Git URL

1. Open Unity Package Manager (Window > Package Manager)
2. Click the **+** button
3. Select **Add package from git URL...**
4. Enter: `https://github.com/yourusername/com.example.myunitypackage.git`
5. Click **Add**

## Package Structure

- `Runtime/NewMonoBehaviourScript.cs` - Sample script
- `Runtime/Testxrh.asmdef` - Assembly definition
